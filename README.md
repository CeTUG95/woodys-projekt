# Programmiersprachen und Konzepte - Projekt

## Haskell-Snake

### Vorraussetzungen:

* GHC (https://www.haskell.org/ghc/)
* cabal (https://www.haskell.org/cabal/)


### Installation:

Fehlende Module installieren:
```bash
$ cabal update
$ cabal install pipes
$ cabal install pipes-concurrency
$ cabal install ansi-terminal
```
